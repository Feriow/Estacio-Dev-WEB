Modos de inserir CSS no HTML:

CSS inline:

CSS Inline

Essa forma implica em declarar o estilo CSS diretamente na tag, no código HTML. A declaração Inline faz uso do atributo “style” procedido por declarações separadas por ponto-e-vírgula “;”. Esse atributo pode ser usado em qualquer tag HTML.

CSS interna:

CSS Interna

Também chamada de CSS Incorporada, é declarada na seção head do documento HTML. A declaração Inline faz uso do atributo “style” procedido por declarações separadas por ponto-e-vírgula “;”. Esse atributo pode ser usado em qualquer tag HTML.

CSS Externa

CSS Externa

Nesse caso, os estilos são declarados em um arquivo externo, com extensão “.css” e vinculados ao documento HTML por meio da tag link ou da diretiva @import dentro da tag head.

CSS em Escopo

CSS em Escopo

Essa forma de aplicação de estilo foi criada a partir da HTML5. Com ela, é possível aplicar estilos no âmbito de escopo, ou seja, específicos para as seções da página onde foram declarados, incluindo os seus elementos filhos.
